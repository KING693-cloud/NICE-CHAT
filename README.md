# NICE Chat

**NICE Chat** is a Progressive Web App (PWA) chat application created by **KING SOLOMON**.  
It allows users to sign up with a phone number, create and join groups, send text, media, and voice messages, and receive push notifications. The app can be installed on Android devices like a native app.

---

## **Features**

- User registration and login via phone number and PIN
- Create and join groups with unique codes
- Real-time group chat
  - Text messages
  - Image, video, and audio messages
  - Voice recording support
- Push notifications for new messages
- Progressive Web App (PWA)
  - Installable on Android devices
  - Offline support with caching
- Group management for admins
- Simple, responsive UI
